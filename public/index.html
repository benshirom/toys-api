"password":"123456" }		 User Sign Up retrun data of user

Toys Routes

# Method URL Body Description
1 get toys-api.onrender.com/toys none get 10 toys
2 get toys-api.onrender.com/toys/search?s={name} none Search toys by name or info get 10 toys
3 get toys-api.onrender.com/toys/category/{category name} none get 10 toys by category name
4 get toys-api.onrender.com/toys/prices?min=70&max=200 none get 10 toys by price can use with min & max or only one of them
must send token
5 post toys-api.onrender.com/toys { "name":"bal11",
"info":"fff",
"category":"sport",
"img":"football.jpg" ,
"price":100 } 		Add toy only img is not required
6 put toys-api.onrender.com/toys/{id of the toy} { "name":"bal11",
"info":"fff",
"category":"sport",
"img":"football.jpg" ,
"price":100 } 		Edit toy only if you created the toy
7 delete toys-api.onrender.com/toys/{id of the toy} none Delete toy only if you created the toy
